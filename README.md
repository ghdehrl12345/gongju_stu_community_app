# BOARD

1. BOARD LIST

- URL : /board/list
- METHOD : GET

2. BOARD DETAIL

- URL : /board/:boardId
- method : GET

3. BOARD CREATE

- URL : /board/create
- METHOD : GER, POST
- body : { title, content, createdAt, UserId}

4. BOARD UPDATE

- URL : /board/update
- METHOD : GET, POST
- body : {id, title, content}

5. BOARD DELETE

- URL : /board/delete
- METHOD : POST
- body : { boardID }
